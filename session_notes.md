# Git Hints
- git status → see changes
- git add . → stage all changes
- git commit -m "message" → commit changes
- don't forget a .gitignore file

# Markdown Hints
- # Heading 1, ## Heading 2
- *italic*, **bold**
- - for lists

# CSS Hints
- Selectors: .class, #id, element
- Colors: hex (#ff0000), rgb(255,0,0)
- Display: block, inline, flex
- Position: static, relative, absolute, fixed
- Useful command: "Ctrl+Shift+C" → inspect element in browser

# Display tips in CSS
- display: block; – element starts on a new line and takes full available width (like <div>, <p>).

- display: inline; – element sits inside a line of text and ignores set width/height (like <span>, <a>).

- display: inline-block; – flows inline but you can set width/height (good for buttons, nav items).

- display: none; – element is not shown at all and takes up no space (often used to hide modals/menus).
